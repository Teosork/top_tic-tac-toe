# TOP_Tic-Tac-Toe

## Introduction 
A classic Tic Tac Toe game built with Vanilla JS.

## Features
1. Custom player names.
2. Responsive 3x3 grid.
3. Win/Draw detection logic.
4. Restart functionality.

### Key Concepts Learned
1. Module Pattern (IIFE).
2. Factory Functions.
3. Closures & Encapsulation.
4. DOM Manipulation.
